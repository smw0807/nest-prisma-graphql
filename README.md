## Description

NestJS API 어플리케이션  
Prisma, GraphQL, MySQL사용

## version

Node : 18.14.1 (npm 9.3.1)  
@nestjs/cli 9.1.4

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
